**运算符（operator）** 是指==**进行运算的动作**==，比如加法运算符+，加法运算符-。
**算子（operand）** 是指==**参与运算的值**==，这个值可能是常数，也可能是变量，还可能是一个方法的返回值。
运算符通常分为==**算数运算符**==和==**关系运算符**==和==**逻辑运算符**==和==**其他**==。

## 00 表达式
[[03 - 1 变量#表达式|表达式]]是一系列运算符和算子（运算对象）的结合，用来完成一次计算。

>[!warning]
>每一个表达式都有一个值，包括含有逻辑和关系运算符的表达式！

*例题   输入两个时间，每个时间分别输入小时和分钟的值，然后输出两个时间之间的差，也以几小时几分表示。*
```c
# include <stdio.h>

int main()
{
	int hour1,minute1;
	int hour2,minute2;
	
	printf("请输入较大的时间：");
	scanf("%d %d",&hour1,&minute1);
	printf("请输入较小的时间：");
	scanf("%d %d",&hour2,&minute2);
	
	int gap1 = hour1 * 60 + minute1;
	int gap2 = hour2 * 60 + minute2;
	
	int	gap = gap1 - gap2;
	
	printf("时间差是%d时%d分",gap/60,gap%60);
	
	return 0;
}
```

## 01 算数运算符
#### （1）算数运算符及优先级

| 优先级 | 运算符  |      运算      |   结合关系   |  举例  |
| :-: | :--: | :----------: | :------: | :--: |
|  0  | ++后缀 |     单目加一     |   自左向右   | a++  |
|  0  | --后缀 |     单目减一     |   自左向右   | a--  |
|  1  | ++前缀 |     单目加一     | ==自右向左== | ++a  |
|  1  | --后缀 |     单目减一     | ==自右向左== | --a  |
|  1  |  +   | ==**单目**==不变 | ==自右向左== | a*+b |
|  1  |  -   | ==**单目**==取负 | ==自右向左== | a*-b |
|  2  |  *   |      乘       |   自左向右   | a*b  |
|  2  |  /   |      除       |   自左向右   | a/b  |
|  2  |  %   |      取余      |   自左向右   | a%b  |
|  3  |  +   |      加       |   自左向右   | a+b  |
|  3  |  -   |      减       |   自左向右   | a-b  |
|  4  |  =   |      赋值      | ==自右向左== | a=b  |
+ 单目：仅有一个算子。
+ 结合关系：即运算顺序。-b的运算顺序即为自右向左，赋予b以负值的属性。

#### （2）复合赋值运算符
+- * / % 可以和赋值运算符=结合起来，形成复合赋值运算符“+=、-=、* =、/=、%=”
```c
a = a + 5;
a += 5;
```
>注意：两个运算符之间不能有空格！
```c
a += (10+100) / 2;
a = a + (10+100) / 2;
a *= 100 + 12;
a = a *(100+12);
```
>注意：复合赋值时，总是==**优先运算等式右边的部分**==再赋值给变量。

#### （3）递增递减运算符 ++ --
“++（递增运算符）”“--（递减运算符）”是两个很特殊的运算符，它们是==**单目运算符**==，算子还==**必须是变量**==。它们的作用就是==**给变量+1或者是-1**==。
```c
int count = 0;
count ++;
count += 1;
count = count + 1;
```

##### 做变量前缀 ++a
此时，++a 为 a ==**递增之后**==的值，a递增+1。（递减同理）
##### 做变量后缀 a++
此时，a++ 为 a ==**递增之前**==的值，a递增+1。（递减同理）

>[!warning]
>注意：++、-- 与变量间可以有空格。

#### （4）除法运算符 / 
对于int整型：/ 会**截断小数部分**，只保留整数部分，不会四舍五入，相当于整除。
对于float等浮点型：则正常运算。

#### （5）取余运算符
C语言规定：C99后，如果第一个运算对象为正数，取余的结果也是正数，如果第二个运算对象为负数，取余的结果也是负数。

>[!tip]
>无论何种情况，只要a 和 b都是整数值，a%b 都可以这么计算：
>**a - （a/b) * b**

## 03 逻辑运算符
#### （1）逻辑非！
结合性：从右往左
a为真，!a为假；反之。
#### （2）逻辑与&& 和 逻辑或 ||
结合性：自左向右
**短路原理**：
```c
if(num2<0&&(num2=num2+1))
```
若num2<0，则直接短路，**num2=num2+1 不会执行**。
## 04 条件运算符
```c
#include <stdio.h> 
int main()
{
	int num1,num2;
	scanf("%d",&num1);
	num2 = num1>0 ? num1 : -num1;
	printf("%d",num2);
	return 0;
}
```
以上是一个输出绝对值的程序，条件运算符由 ？和：组成，？前是表达式，表达式为真，整个条件运算符运算后的值为第一个值，若为假，则为第二个值。
