for循环的基本格式：
```c
for(initialize;test;update)
	statement
for(count=1;count<=101;count++)
	...
```
for循环将（初始化；测试；更新）组合在一起。

### ==01 for循环的9种用法==
##### （1）使用递减运算符来递减计数器
```c
#include <stdio.h>
int main()
{
	int secs;
	
	for (secs = 5; secs > 0; secs--)
		printf("%d seconds!\n",secs);
	printf("We have ignition!\n");
	return 0;
}
```
