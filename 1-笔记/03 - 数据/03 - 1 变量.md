变量是一个保存数据的地方，变量有自己的数据类型，且可更改变化。这是计算机中数据的一种表现形式。
___
## 01 定义变量
```c
#include <stdio.h>
int main()
{
	int price = 0;
	//这一行定义了一个变量price，初始值是0，类型是int
}
```
>[!tip]
>注意：int price = 0 ; 这一行定义了一个==**变量price，初始值是0，类型是int**==

### **变量定义的一般形式**
<类型名称><变量名称> ==;==
例如：int price ; int amount ; ==int price , amount ; **一行定义多变量**==
### **变量的名字**
变量的名字是一种“标识符”
==**基本规则：**==
+ 标识符只能由**字母、数字和下划线**组成
+ 数字不能出现在第一个位置上
+ C语言的关键字（保留字）不能用做标识符

### ANSI C 与 C99 的差异
```c
// C99
int price = 0;
printf("请输入金额：");
scanf("%d",&price);
int change = 100 - price;
printf("找您%d元。\n",change);

// ANSI C
int price = 0;
int change = 0;
printf("请您输入金额：");
scanf("%d",&price);
change = 100 - price;
printf("找您%d元。\n",change);
```
C99可在定义变量时做计算，而ANSI C只能先在代码开头定义变量，后做计算。

## ==02 变量的赋值和初始化==
```c
#include <stdio.h>
int main()
{
	int price = 0;
}
```
这个式子中 = 是一个赋值运算符，将等号右边的值赋给左边

### 初始化
当赋值发生在变量的时候，即为变量的初始化。
C语言并不强制要求变量在定义时进行初始化，但是==**变量在使用时必须进行初始化，否则变量的值会随机乱码**==。
```c
int price = 0;
int price = 0,amount = 0;
```
>[!warning]
>注意：当组合初始化时，多变量必须==**分开单独赋值**==。
```c
int a;
int b;
scanf("%d %d",&a,&b);
```
>[!tip]
>注意：scanf==**成功**==读取整数赋值给变量a,b也是对a,b的初始化。
### 表达式
有运算符的式子就叫做表达式。是一系列运算符和算子的结合，用来完成一次计算。

## ==03 变量类型==
C语言是一个有类型的语言，所有的变量在使用之前都必须定义或声明，即==**所有的变量都有一个确定的数据类型，且不能更改**==。数据类型决定了该变量能存放什么类型的数据。

### （1）整型类型 int
==**两个整数做运算的结果只能是整数**==
```c
int a = 3;
int b = 50;
printf("%d",b/a);
```
输出结果是 16 ，==**仅取整数部分**==。

其它整数类型：
+ short int （或简写为short）**常用于较小数值，且占用的存储空间比int少**
+ long int（或简写为long）**长整型，适合较大数值**
+ long long int（或简写为long long）**占用的存储空间更多，该类型至少占64位**
+ unsigned int（或简写为unsigned）**只适用于非负值的场合**，无符号整型可以表示更大的数。

>[!note]
>补充：[[03 - 2 常量#^cf640e|char类型]]用来储存字符常量，实际上存储的是字符对应的ASCII码值，因此也是用int型来存储的。

### （2）浮点类型 float \ double
带小数点的数值。浮点的本意就是指小数点是浮动的，是计算机内部表达非整数（包括分数和无理数）的一种方式。

>[!tip]
>注意：当浮点数与整数一起运算时，==**C语言会将整数转化为浮点数**==。

double用来代表==**双精度浮点数**==，float用来代表==**单精度浮点数**==。双精度浮点数采用更大的存储空间，从而可以容纳更多的小数部分和指数部分，增大浮点数能表示的数值范围。
>[!tip]
>一般说来，`float` 有效数字约 6~7 位，`double` 约 15~17 位。
