固定不变的数，是常数。直接写在程序里，称作直接量（literal）。
更好的方式，是定义一个常量。这是计算机中数据的一种表现形式。
___
## 01 定义常变量
```c
const int AMOUNT = 100;
```
定义常变量（read-only variable）可以赋予数据更生动形象的名字，便于理解。

#### 修饰符 const 
const 是一个**修饰符**，**加在int的前面**，用来给变量添加一个“不变的”属性。当添加const属性后，这个变量的值==**一旦初始化，就不能被修改了**==。
>[!note]
>注意：添加const属性的变量，变量名一般是**全大写**的。

## 02 常量分类
#### （1）整型常量
如1000，0，-123等整数，都是整型常量。
>[!note]
>十进制 Dec  八进制 Oct  十六进制 Hex
#### （2）实型常量
+ ==**十进制小数形式**==
	由数字和小数点组成。（就是小数）
+ ==**指数形式**==
	如12.34e3（代表12.34乘10的三次方）、12e3、12.e3、.45e3（e或E都可）
> [!warning]
> 注意：e或E前面必须有数字，且后面必须是整数。
#### （3）字符常量
+ ==**普通字符**==

	1.用==**单撇号括起来的一个字符**==。如'a','b','#','?','3'。==不能写成'ab'或'12'==

	2.单撇号只是界限符，字符常量只能是一个字符，==不包括单撇号==。

	3.普通字符存储在计算机的存储单元中，是==以其代码（ASCII）存储的==。
>[!note]
> **补充：char类型**是用于储存字符的变量类型，但从技术层面看，char是整数类型。因为char实际上储存的是整数而不是字符，确切的来说，是**该字符对应的ASCII码值**。 ^cf640e
+ ==**转义字符**==
	一种特殊形式的字符常量，就是以字符“\”开头的字符序列。这是一种==在屏幕上无法显示的“控制字符”==，在程序中也无法用一个一般形式的字符来表示，只能采用这种特殊的形式表示。 ^fbd6bc

**转义字符及作用**

| 转义字符                        | 字符值                       | 输出结果                 |
| --------------------------- | ------------------------- | -------------------- |
| \\'                         | 一个单撇号(')                  | 输出单撇号字符 '            |
| \\''                        | 一个双撇号('')                 | 输出双撇号字符"             |
| \\?                         | 一个问号(?)                   | 输出问号字符?              |
| \\\                         | 一个反斜线(\\)                 | 输出反斜线字符\             |
| \a                          | 警告(alert)                 | 产生声音或视觉信号            |
| \b                          | 退格(backspace)             | 将光标当前位置后退一个字符        |
| \f                          | 换页(form feed)             | 将光标当前位置移到下一页的开头      |
| \n                          | 换行                        | 将光标当前位置移到下一行的开头      |
| \r                          | 回车(carriage return)       | 将光标当前位置移到本行的开头       |
| \t                          | 水平制表符                     | 将光标当前位置移到下一个Tab键位置   |
| \v                          | 垂直制表符                     | 将光标当前位置移到下一个垂直制表对齐点  |
| \o、\oo、\ooo<br>其中o代表一个八进制数字 | 与该八进制码对应的ASCII==**字符**==  | 与该八进制码对应的==**字符**==  |
| \xh<br>其中h代表一个十六进制数字        | 与该十六进制码对应的ASCII==**字符**== | 与该十六进制码对应的==**字符**== |
>[!note]
>\b 转义字符，只有在其后有**普通字符**时，才会触发移动光标和**覆盖**的效果。\b做的只能是**覆盖**，而不是删除！！
#### （4）字符串常量
如"boy","123"等，用双撇号把若干个字符括起来。字符串常量是双撇号中的全部字符，不包括双撇号。
>[!warning]
>注意：==**单撇号内只能包含一个字符**==，构成普通字符，双撇号内可以包含一个字           符串。
#### （5）符号常量
用#define指令，指定用一个符号名称代表一个常量。
```c
#include <stdio.h>
#define PI 3.1415926   //预编译处理阶段
int main()
{
	double a = PI         //此时a的类型是浮点型
	printf("%f",a);    
	return 0;
}
```
+ define定义的宏常量
	宏常量与const变量有区别。宏常量只是一个代号。
	
	定义的宏常量在==**预编译阶段**==完成==**文本替换**==。替换后赋值给变量后，其数据类型依赖于变量的数据类型。

| 特性    | `#define` 宏常量         | `const` 变量    |
| ----- | --------------------- | ------------- |
| 处理阶段  | ==**预处理阶段（文本替换）**==   | ==**编译阶段**==  |
| 类型检查  | 无类型检查                 | 有严格的类型检查      |
| 内存占用  | 不分配内存（展开到代码中）         | 分配内存（可存放在只读段） |
| 调试可见性 | 不可见（宏已被替换）            | 可见（调试器可查看）    |
| 作用域   | 从定义处到文件末尾（或 `#undef`） | 遵循 C 语言作用域规则  |

## 03 常量类型转换
这种类型的转换不是强制的，需要转换相应类型的值。
#### （1）一个常量后加u或U
一个整常量后面加一个字母u或U，认为是unsigned int型，如12345u，在内存 中按unsigned int规定的方式存放。
#### （2）一个常量后加l或L
在一个整常量后面加一个字母l或L，则认为是long int型常量。 例如： 123l.432L.0L。
