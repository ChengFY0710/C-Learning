while循环的基本格式：
```c
while(a == 1)
{
	a=a+1;
	printf("%d",a);
}
while(b == 1)
	printf("%d",b);
```
---
## 01 语句要点
while语句循环的内容，若是简单语句，到第一个分号处结束，若是复合语句，到最后一个花括号处结束。
## 02 真值
对于C而言，表达式为真的值为1，表达式为假的值为0。
```c
while(1)
{
	......
}
```
该循环会一直执行。
>[!note]
>1 和 0 可以作为while语句的测试表达式。一般而言，所有的非零值都视为真，只有0被视为假。

## 03 循环辅助 break和continue
break语句会终止包含它的循环，continue会终止包含它的**本轮**循环。
