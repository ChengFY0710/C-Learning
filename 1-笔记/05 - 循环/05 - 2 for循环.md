for循环的基本格式：
```c
for(initialize;test;update)
	statement
for(count=1;count<=101;count++)
	...
```
for循环将（初始化；测试；更新）组合在一起。

---
## 01 for循环的几种用法
#### （1）使用递减运算符来递减计数器
```c
#include <stdio.h>
int main()
{
	int secs;
	
	for (secs = 5; secs > 0; secs--)
		printf("%d seconds!\n",secs);
	printf("We have ignition!\n");
	return 0;
}
```
#### （2）用字符代替数字计数
```c
for(ch='a'; ch<='z'; ch++);
```

## 02 for循环的注意事项
#### （1）for循环的格式
```c
for(exp1;exp2;exp3)语句
```
其中，==**exp1可以省略**==，即可以不设初始值。
exp2其实也可以省略，但是此刻因缺少循环条件表达式，循环会==**无休止的进行下去**==，也就是认为表达式2始终为==**真**==。
exp3也可以省略，此时需要重新设置使循环结束的条件。
```c
for( ; ;)printf("Hello World!");
相当于
while(1)printf("Hello World");
```
exp1可以是设置循环变量初值的赋值表达式，也可以是==**与循环变量无关的其他表达式**==。
```c
for(i=1,s=2;i<5;i++,s++) ;
```
exp1和exp3可以使用==**逗号运算符**==，包含多个表达式。
#### （2）for循环的执行顺序
```c
for(i=1,j=1;i<=100;i++,j++) ;
```
循环结束后，i,j的值为101，不是100！
for先执行exp1，再执行exp2，判断为真，执行语句，语句执行结束，执行exp3，然后再执行exp1...
