**函数**是完成特定任务的**独立程序代码单元**。语法规则定义了函数的结构和使用方式。
一般而言，函数有**执行某些动作**和**找出一个值供某些程序使用**两种功能。
使用函数的优势：
+ 避免编写重复代码。
+ 是程序更加模块化，提高代码可读性。

---

## 01 创建并使用简单函数
```c
#include <stdio.h>
#define NAME "GIGATHINK, INC."
#define ADDRESS "101 Megabuck Plaza"
#define PLACE "Megapolis, CA 94904"
#define WIDTH 40
void starbar(void);  /*函数原型*/
int main(void)
{
    starbar();
    printf("%s\n", NAME);
    printf("%s\n", ADDRESS);
    printf("%s\n", PLACE);
    starbar();       /*函数调用*/    
    return 0;
}
void starbar(void)   /*函数定义*/
{
    int count;    
    for (count = 1; count <= WIDTH; count++)
        putchar('*');
    putchar('\n');
}
```
#### （1）函数原型
函数和变量一样，有多种类型。任何程序在使用函数之前都要先声明该函数的类型。因此，在main()函数定义的前面出现了下面的ANSI C风格的函数原型：
```c
void starbar(void);
```
圆括号代表starbar是一个**函数名**，第一个void是**函数类型**（void类型表示函数没有返回值），第二个void表明该函数不带**参数**。**分号**表明这是在声明函数。
**函数原型**声明了一个函数，并告诉编译器在**别处查找函数的的定义**。
