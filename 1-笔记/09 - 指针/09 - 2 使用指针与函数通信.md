若要实现两个数字交换的函数，只将实际参数输入形式参数后，并没有交换主调函数mian函数中的变量的值。

因此可以借助于指针和地址，来实现两个函数间的通信。

---

```c
#include <stdio.h>

void exchange(int * ,int * );

int main()
{
    int a = 5;
    int b = 10;
    printf("Originally , a=%d , b=%d\n", a, b);
    exchange(&a, &b);
    printf("Exchanged , a=%d , b=%d", a, b);

    return 0;
}

void exchange(int * a, int * b)
{
    int temp;
    temp = *a;
    *a = *b;
    *b = temp;
}
```
